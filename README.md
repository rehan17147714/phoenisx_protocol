## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Copy `.env.example` to `.env` and add your DeepSeek API key:
   ```
   VITE_DEEPSEEK_API_KEY=your_deepseek_api_key_here
   ```
4. Start development server: `npm run dev`
